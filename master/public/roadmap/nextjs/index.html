Roadmap Belajar Next.js

Panduan lengkap dari pemula hingga mahir untuk menguasai Next.js.

Tahapan Belajar

1. Pengenalan Dasar

  • Apa itu Next.js: Framework React untuk rendering server-side dan static-site generation.
  • Mengapa Memilih Next.js: SEO-friendly, performa tinggi, dan mendukung fitur modern seperti API routes.
  • Persiapan Lingkungan: Install Node.js & npm/yarn. 
    npx create-next-app@latest
    cd nama-proyek
    npm run dev

2. Struktur Folder dan File

  • Penjelasan Struktur Dasar:
    pages/: Routing berbasis file.
    public/: File statis (gambar, font, dll).
    styles/: Styling global atau modular.
  • File Penting: next.config.js, _app.js, _document.js.

3. Dasar-Dasar Next.js

  • Routing dan Navigasi: Routing otomatis di folder pages/.
  • Dynamic Routing:
    // pages/[id].js
export async function getServerSideProps({ params }) {
  return { props: { id: params.id } };
}
  • Rendering: CSR, SSR, SSG, ISR.

4. API Routes

  • Membuat API dengan Next.js: Folder pages/api.
  • Contoh Endpoint API: 
    export default function handler(req, res) {
  res.status(200).json({ message: "Hello World" });
}

5. Data Fetching

  • Server-Side Rendering (SSR): Fungsi getServerSideProps.
  • Static-Site Generation (SSG): Fungsi getStaticProps & getStaticPaths.
  • Client-Side Fetching: Menggunakan fetch atau library seperti Axios/SWR.

6. Styling

  • CSS Modules: Penamaan unik untuk komponen.
    import styles from './Home.module.css';
  • Styled JSX (default Next.js).
  • Integrasi Framework CSS: Tailwind CSS atau Bootstrap.

Siap Belajar Next.js?

Mulailah perjalanan Anda sekarang dengan membuat proyek kecil untuk mengaplikasikan konsep-konsep di atas.
